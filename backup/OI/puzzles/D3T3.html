Description

代数课上，老师写下了两个向量$\vec a$，$\vec b$，并且允许小v任意地重排某一个向量的所有元素。定义$x=\vec a\cdot \vec b$，老师让小v求出重排之后，$x$所能达到的最大值是多少。

Input Format

输入一共三行。

第一行是一个整数$n$，代表向量的维数。

第二、三行分别为向量$\vec a,\vec b$的各个元素。

Output Format

一个整数，即$x$的最大值。
