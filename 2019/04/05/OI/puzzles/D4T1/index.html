D4T1.小v选课表

发表于 2019-04-05 | 评论数：

Description

又到了夏季学期选课的时间，小v打算多选一些课来拯救一下自己的GPA。小v选课时只考虑一个因素，即课程之间会不会冲突。小v把不同课程的持续时间抽象为若干个开区间，他必须保证选的课程对应的时间没有交集。假设每门课程只对应一个区间，那么小v最多能选几门课呢？

Input Format

第一行一个整数$n​$，表示课程总数。

接下来$n$行，每行两个数，表示课程的开始时间$t1_i$和结束的时间$t2_i$。

Output Format

一个数字，表示小v能选课程的最大数目。

Sample Output

Hint

样例选取的课程区间为：$(0, 3)$, $(4, 8)$, $(9, 11)$, $(12, 15)$, $(15, 20)$。

Data Range

$n\le 10^5$，$0\le t1_i \lt t2_2 \le 10^6$。

思路

这是一道基础的贪心题目。我们可以先对区间的右端点排序，依次检查左端点是否在前一个区间右端点的左方。如果是，则该区间舍弃；否则选择该区间，并更新用于比较的右端点的坐标。
