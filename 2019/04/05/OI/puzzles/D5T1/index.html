D5T1.小v吃青团

发表于 2019-04-05 | 评论数：

Description

又到了吃青团的时节。小v来到食堂，发现里面整整齐齐地放着一整排的青团，食堂阿姨斜眼微笑和蔼地说，只要小v能解决一个数学问题，就把这些青团都送给他。一共有$n$个青团，第$i$个青团位于位置$x_i$处，且每个青团都在不同的位置。所有的青团构成集合$A$，记函数$F(a)=\max _{i,j\in a}|x_i-x_j|$，其中$a$是$A$的一个非空子集。食堂阿姨向小v问道：$\sum_{a\subseteq A,a\neq \varnothing}F(a)$的结果是多少。~~为了让他长胖，快帮小v解决这个问题吧。

Input Format

第一行，有一个整数$n$，表示青团的数目。

第二行，有$n​$个整数，代表青团的坐标。

Output Format

输出一行，即上式的结果。由于结果可能非常大，故输出结果对$10^9+7​$取模的值。

Hints

在样例2中，满足条件的$a​$为$\{4,3\}​$，$\{4,1\}​$，$\{3,1\}​$，$\{4,3,1\}​$。故结果为$(4−3)+(4−1)+(3−1)+(4−1)=9​$。

Data Range

$1\le n \le 10^5$，$1\le x_i\le 10^9​$。

思路

这道题本身不难想，但是在OJ上用暴力的方法很容易被卡时间，所以下面的代码用了一个优化。排序之后，遍历到第$i$个元素时，与前$i-1$个元素可构成$i-1$个新的集合，打表存储之前的总和，可以得到递推关系。
